## 🔧 ARCHITECTURE TECHNIQUE

### Machine à états Campagnes

```
BROUILLON ──schedule──→ PLANIFIEE ──send──→ EN_COURS ──auto──→ TERMINEE
    │                      │                    │
    │                      │                    ├─pause──→ PAUSE
    │                      │                    │             │
    │                      │                    │             └─resume──→ EN_COURS
    │                      │                    │
    │                      └─cancel─────────────┴──→ ANNULEE
    │
    └─delete (allowed)
    └─send (direct) ──────────────────────────────→ EN_COURS → TERMINEE
```

**Transitions validées** :
- BROUILLON : update/delete/schedule/send autorisés
- PLANIFIEE : cancel/send autorisés
- EN_COURS : pause/cancel autorisés
- PAUSE : resume/cancel autorisés
- TERMINEE/ANNULEE : fin de vie (lecture seule)

### Machine à états Scenarios

```
INACTIF ──activate──→ ACTIF ──deactivate──→ INACTIF
   │                     │                      │
   │                     │                      └─archive──→ ARCHIVE
   │                     │
   └─delete (allowed)    └─execute (allowed)
```

### Triggers Scenarios (7 types)

1. **NOUVEAU_CLIENT** : Déclenché à création client (email bienvenue)
2. **ANNIVERSAIRE** : Déclenché jour anniversaire client
3. **DATE_ECHEANCE** : Déclenché X jours avant échéance (rappels)
4. **ACTION_CLIENT** : Déclenché sur action spécifique (signature contrat, etc.)
5. **INACTIVITE** : Déclenché après X jours sans activité (relance)
6. **MANUEL** : Exécution manuelle uniquement (via API)
7. **WEBHOOK** : Déclenché par webhook externe

```typescript
// .env
RESEND_API_KEY=re_xxx

// Dans CampaignService.sendCampaign()
import { Resend } from 'resend'
const resend = new Resend(process.env.RESEND_API_KEY)

for (const recipient of campaign.recipients) {
  const { id } = await resend.emails.send({
    from: campaign.fromEmail,
    to: recipient.email,
    subject: campaign.subject,
    html: campaign.htmlContent,
    headers: {
      'X-Entity-Ref-ID': recipient.id, // Pour webhooks
    },
  })
  
  await prisma.campaignRecipient.update({
    where: { id: recipient.id },
    data: {
      providerMessageId: id,
      status: 'ENVOYE',
      sentAt: new Date(),
    },
  })
}
```

```typescript
export async function POST(req: Request) {
  const event = await req.json()
  
  const recipient = await prisma.campaignRecipient.findFirst({
    where: { providerMessageId: event.messageId }
  })
  
  if (event.type === 'email.delivered') {
    await prisma.campaignRecipient.update({
      where: { id: recipient.id },
      data: { status: 'DELIVRE', deliveredAt: new Date() }

  if (event.type === 'email.opened') {
    await prisma.campaignRecipient.update({
      where: { id: recipient.id },
      data: {
```
